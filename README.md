# meiomap
#Requirements
numpy
pandas
argparse

#Installation
git clone https://github.com/hreffie/meiomap.git

#Description 
A heuristic approach to the pilot stage of phasing. Output is in the format of b bed files:
chr	start	stop	phase.

#Usage
./meiomap.py --infile <infile> --outdir <outdir>
./meiomap.py -h
